To run this program you will first need to install chromedriver from the following link:
https://chromedriver.storage.googleapis.com/index.html?path=2.33/
You will also need to install selenium package for python

After you have installed chromedriver for your machine you will need to type your username and password into the input_data.txt file 
Place the openbrowser.py, chromedriver.exe, and input_data.txt into one directory
Open openbrowser.py and change the webdriver.Chrome line by placing in the path to the chromedriver.exe
After this is completed you may run the program
